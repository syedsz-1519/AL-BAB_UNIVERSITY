Al-Bab Islamic University (جامعة الباب الإسلامية)
A premium, interactive, state-of-the-art scholastic platform integrating classical Islamic knowledge (Quran, Hadith, Fiqh) with cutting-edge modern interactive visualizations, cognitive-behavioral recovery tools, and scholastic database portal systems.

🌟 Main Sections
🪐 1. Celestial Globe (Verse Explorer & Scholastic Hubs)
An interactive revolving 3D globe detailing coordinates and histories of verified classical scholastic hubs (Medina, Bukhara, Baghdad, Cairo, Fez, Cordoba) paired with Quran verse commentary and maps.

📚 2. Course Sections (Scholastic Universe)
Comprehensive curriculum explorer categorizing Islamic sciences—including Quranic Tafsir, Prophetic Traditions (Hadith), Jurisprudence (Fiqh), Dialectic Theology (Kalam), and Civic Ethics (Mu'amalat).

🕌 3. Islamic Five Pillars Module
An interactive, high-fidelity visualization detailing the theological and practical frameworks of the Five Pillars of Islam (Shahada, Salah, Zakah, Sawm, and Hajj) with authentic legal evidence.

📿 4. Dhikr & Devotional Checklist
A personal digital spiritual planner featuring authentic morning and evening supplications (Adhkar), complete with translation, audio-loop timings, and interactive trackers.

📖 5. Hadith on Ilm (Canonical Search)
An advanced canonical search interface focused on prophetic traditions highlighting the virtue of seeking knowledge (Ilm), including full chain checks (Isnad), verified grading (Sahih, Hasan), and source texts.

🎓 6. Student & Admin Sanctum Portals
A multi-user management hub linking students and administration:

Student Dashboard: View notices, submit graded assignments, and inspect registrations.
Admin sanctum: Register admissions, post notices, grade assignments, and monitor real-time transaction ledger.
🔮 Upcoming Modules
AI Scholastic Bot: Inline integration of the Gemini-driven curricular tutor.

Classical Munazara Room: Interactive debating arena simulating historic theological dialectics.
🚀 Running Locally
Prerequisites
Node.js (v18 or higher)
npm or yarn
1. Clone & Install Dependencies
bash

git clone https://github.com/syedsz-1519/AL-BAB_UNIVERSITY.git
cd AL-BAB_UNIVERSITY
npm install
2. Configure Environment Variables
Create a .env file in the root directory:

env

GEMINI_API_KEY=your_gemini_api_key_here
3. Start Development Server
bash

npm run dev
Open http://localhost:3000 in your browser.
